## Getting Started

### Installation

1. Clone the repo

   ```sh
   https://github.com/leonardodiasb/blog-app.git
   ```

### Usage

1. Install ruby gems  `bundle install`

2. Install node `npm install`

3. Run the web server `rails s`

4. Visit http://localhost:3000/ in your browser!


## Acknowledgements
